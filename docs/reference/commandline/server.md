# hbm server
***

```markdown
Launch the HBM server

Usage:
  hbm server [flags]
```

## Examples

```bash
# hbm server
INFO [0000] Server has completed initialization
INFO [0000] HBM server         logdriver=standard storagedriver=sqlite version=0.8.0
INFO [0000] Listening on socket file
```
