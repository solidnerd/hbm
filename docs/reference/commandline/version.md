# hbm version
***

```markdown
Show the HBM version information

Usage:
  hbm version [flags]
```

## Examples

```bash
# hbm version
Version:     0.8.0
Git commit:  HEAD
Built:       1970-01-01 00:00:00 +0000 UTC
Go version:  go1.8.3
OS/Arch:     linux/amd64
```
